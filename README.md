# Projek UTS Mobile Apps

## Deskripsi

Projek ini merupakan aplikasi mobile yang dibuat untuk memenuhi tugas UTS.
Aplikasi ini menampilkan implementasi dasar pengembangan mobile, termasuk tampilan antarmuka dan interaksi pengguna.

## Fitur

* Halaman login dan registrasi
* Navigasi antar halaman
* Tampilan UI sederhana dan responsif
* Struktur project rapi dan modular

## Teknologi yang Digunakan

* Android Studio
* Java atau Kotlin
* XML untuk desain UI

## Cara Menjalankan Project

1. Clone repository:
   https://github.com/iqbalhidayatur/ProjekUTS-MobileApps.git

2. Buka di Android Studio

3. Tunggu proses Gradle selesai

4. Jalankan di emulator atau device fisik

* app/ → source code utama aplikasi
* gradle/ → konfigurasi build
* assets/ atau res/ → resource seperti gambar dan layout
